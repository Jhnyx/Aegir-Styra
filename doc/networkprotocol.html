Aegir Styra Network Documentation

The networking protocol used in Aegir Styra 5 has not yet been documented in detail. Aegir Styra uses UDP port 18304 by default, and uses the enet networking library.

If you want to write a program to interface with Aegir Styra via its networking protocol, please see the source code of networkPrimary.cpp (used when Aegir Styra is the master sending scenario information) and networkSecondary.cpp (used when Aegir Styra is receiving scenario information, as a secondary display), or contact james@aegirstyra.co.uk.
